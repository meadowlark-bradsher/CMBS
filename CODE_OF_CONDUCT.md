# Code of Conduct

This project adopts the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)
as its code of conduct. All participants — maintainers, contributors,
and users interacting in project spaces (issues, pull requests,
discussions) — are expected to abide by it.

## Reporting

To report a concern, please open a [private security advisory](https://github.com/meadowlark-bradsher/CMBS/security/advisories/new).
Reports are reviewed by the project maintainers and handled
confidentially.

## Scope

This applies to all project-controlled spaces and to behavior that
represents the project in public spaces (e.g., posting from an
official account, speaking at an event as a project representative).

## Enforcement

Maintainers follow the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines)
when deciding on consequences for violations.
